# Invastor
Delivery system for App and Web similar to Uber

<img src="logo.jpg">

## App (Client & Driver)
1. React-Native version: `0.5.7`
2. Communicate server using WebSocket
3. FCM for Notification
4. Social login using Google and Facebook
